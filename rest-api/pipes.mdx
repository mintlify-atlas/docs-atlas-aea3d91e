---
title: "Pipe Endpoints"
description: "REST API endpoints for managing Meerschaum pipe attributes and data."
---

All pipe endpoints are prefixed with `/mrsm/pipes`.

Pipes are identified by three keys in the URL path:
- `{connector_keys}` — e.g. `sql:main`
- `{metric_key}` — e.g. `weather`
- `{location_key}` — e.g. `nyc` or `[None]` for no location

## List All Pipes

<ResponseField name="GET /mrsm/pipes" type="object">
  Return all registered pipes as a nested dictionary keyed by connector, metric, and location.
</ResponseField>

**Required scope:** `pipes:read`

<CodeGroup>
```bash curl
curl http://localhost:8000/mrsm/pipes \
  -H "Authorization: Bearer <token>"
```

```python Python
import requests

resp = requests.get(
    "http://localhost:8000/mrsm/pipes",
    headers={"Authorization": "Bearer <token>"},
)
pipes = resp.json()
```
</CodeGroup>

**Query parameters:**

<ParamField query="connector_keys" type="string">
  Filter by connector key.
</ParamField>

<ParamField query="metric_keys" type="string">
  Filter by metric key.
</ParamField>

<ParamField query="location_keys" type="string">
  Filter by location key.
</ParamField>

## Get Pipe Keys

## Register a Pipe

<ResponseField name="POST /mrsm/pipes/{connector_keys}/{metric_key}/{location_key}/register" type="SuccessTuple">
  Register a new pipe. Returns `[true, "..."]` on success or `[false, "..."]` on failure.
</ResponseField>

**Required scope:** `pipes:write`

<CodeGroup>
```bash curl
curl -X POST \
  "http://localhost:8000/mrsm/pipes/sql:main/weather/nyc/register" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"columns": {"datetime": "timestamp"}, "tags": ["prod"]}'
```

```python Python
import requests

resp = requests.post(
    "http://localhost:8000/mrsm/pipes/sql:main/weather/nyc/register",
    headers={"Authorization": "Bearer <token>"},
    json={"columns": {"datetime": "timestamp"}, "tags": ["prod"]},
)
print(resp.json())  # [True, "Success"]
```
</CodeGroup>

<ParamField query="instance_keys" type="string">
  Instance connector key (defaults to the server's configured instance).
</ParamField>

<ParamField body="parameters" type="object">
  Optional pipe parameters dictionary to set on registration.
</ParamField>

## Edit Pipe Parameters

  Update an existing pipe's parameters.

curl -X PATCH \
  "http://localhost:8000/mrsm/pipes/sql:main/weather/nyc/edit" \

<ParamField query="patch" type="boolean" default="false">
  If `true`, merge the new parameters into existing ones instead of replacing.
</ParamField>

## Sync Data into a Pipe

<ResponseField name="POST /mrsm/pipes/{connector_keys}/{metric_key}/{location_key}/data" type="SuccessTuple">
  Add rows to an existing pipe.
</ResponseField>

**Required scope:** `pipes:write`

<CodeGroup>
```bash curl (JSON)
curl -X POST \
  "http://localhost:8000/mrsm/pipes/sql:main/weather/nyc/data" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '[
    {"timestamp": "2026-01-01", "id": 1, "value": 100.1},
    {"timestamp": "2026-01-02", "id": 1, "value": 200.2}
  ]'
```

```python Python
import requests

rows = [
    {"timestamp": "2026-01-01", "id": 1, "value": 100.1},
    {"timestamp": "2026-01-02", "id": 1, "value": 200.2},
]
resp = requests.post(
    "http://localhost:8000/mrsm/pipes/sql:main/weather/nyc/data",
    headers={"Authorization": "Bearer <token>"},
    json=rows,
)
print(resp.json())  # [True, "Synced 2 rows"]
```
</CodeGroup>

<ParamField query="check_existing" type="boolean" default="true">
  Deduplicate against existing rows before inserting.
</ParamField>

<ParamField query="blocking" type="boolean" default="true">
  Wait for the sync to complete before returning.
</ParamField>

<ParamField query="workers" type="integer">
  Number of worker threads for parallel operations.
</ParamField>

## Get Pipe Data

<ResponseField name="GET /mrsm/pipes/{connector_keys}/{metric_key}/{location_key}/data" type="string (JSON)">
  Return pipe data as a JSON-encoded array of records.
</ResponseField>

**Required scope:** `pipes:read`

```bash
curl "http://localhost:8000/mrsm/pipes/sql:main/weather/nyc/data?begin=2026-01-01&limit=100" \
  -H "Authorization: Bearer <token>"
```

<ParamField query="begin" type="string | int">
  Filter rows newer than or equal to this value (ISO 8601 or integer).
</ParamField>

<ParamField query="end" type="string | int">
  Filter rows older than this value.
</ParamField>

<ParamField query="params" type="string">
  JSON-encoded filter parameters dictionary.
</ParamField>

<ParamField query="select_columns" type="string">
  JSON-encoded list of columns to return.
</ParamField>

<ParamField query="omit_columns" type="string">
  JSON-encoded list of columns to exclude.
</ParamField>

<ParamField query="limit" type="integer">
  Maximum number of rows to return (capped by server configuration).
</ParamField>

<ParamField query="order" type="string" default="asc">
  Sort order: `asc` or `desc`.
</ParamField>

<ParamField query="date_format" type="string" default="iso">
  Datetime serialization format: `iso` (ISO 8601) or `epoch` (milliseconds).
</ParamField>

<ParamField query="date_unit" type="string" default="us">
  Timestamp precision: `s`, `ms`, `us`, or `ns`.
</ParamField>

<ParamField query="double_precision" type="integer" default="15">
  Decimal places for floating-point serialization (maximum `15`).
</ParamField>
