---
title: "Jobs Endpoints"
description: "REST API endpoints for managing Meerschaum background jobs and streaming logs."
---

All job endpoints are prefixed with `/mrsm/jobs`.
Log endpoints are prefixed with `/mrsm/logs`.

## List Jobs

<ResponseField name="GET /mrsm/jobs" type="object">
  Return metadata for all current jobs.
</ResponseField>

**Required scope:** `jobs:read`

<CodeGroup>
```bash curl
curl http://localhost:8000/mrsm/jobs \
  -H "Authorization: Bearer <token>"
```

```python Python
import requests

resp = requests.get(
    "http://localhost:8000/mrsm/jobs",
    headers={"Authorization": "Bearer <token>"},
)
for name, meta in resp.json().items():
    print(name, meta["status"])
```
</CodeGroup>

**Example response:**

```json
{
  "happy_seal": {
    "sysargs": ["sync", "pipes", "--loop"],
    "result": null,
    "restart": false,
    "status": "running",
    "daemon": {
      "status": "running",
      "pid": 12345,
      "properties": {}
    }
  }
}
```

## Create a Job

<ResponseField name="POST /mrsm/jobs/{name}" type="SuccessTuple">
  Create and start a new background job.
</ResponseField>

**Required scope:** `jobs:execute`, `jobs:write`

<CodeGroup>
```bash curl
curl -X POST \
  http://localhost:8000/mrsm/jobs/my-sync-job \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '["sync", "pipes", "--loop"]'
```

```python Python
import requests

resp = requests.post(
    "http://localhost:8000/mrsm/jobs/my-sync-job",
    headers={"Authorization": "Bearer <token>"},
    json=["sync", "pipes", "--loop"],
)
print(resp.json())  # [True, "Started job 'my-sync-job'."]
```
</CodeGroup>

<ParamField body="metadata" type="array | object" required>
  Either a list of sysargs (e.g. `["sync", "pipes", "--loop"]`) or a dictionary with `sysargs` and optional `properties` keys.
</ParamField>

**Example response:**

```json
[true, "Started job 'my-sync-job'."]
```

<Note>
  If a job with the given name already exists, the server returns HTTP `409 Conflict`.
</Note>

## Stream Logs (WebSocket)

<ResponseField name="WS /mrsm/logs/{name}/ws" type="WebSocket">
  Stream log output from a job in real time over a WebSocket.
</ResponseField>

Authentication is performed by sending the bearer token as the first WebSocket message.

```python
import asyncio
import websockets

async def stream_logs():
    uri = "ws://localhost:8000/mrsm/logs/happy_seal/ws"
    async with websockets.connect(uri) as ws:
        # First message must be the bearer token
        await ws.send("eyJhbGci...")
        async for message in ws:
            print(message)

asyncio.run(stream_logs())
```

The server sends:
- Log lines as plain text strings
- A stdin prompt message when the job is blocking on input
- A stop message followed by the `SuccessTuple` result when the job finishes
